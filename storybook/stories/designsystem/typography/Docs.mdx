# Типографика

Шрифты есть в двух вариантах:
- [**desktop**](https://www.figma.com/file/vHpOsYnsrGbKRvZP6MEPal/%F0%9F%92%A0Tokens-%7C-Typography-%7C-Desktop) — для большого веба
- [**mobile**](https://www.figma.com/file/78IzDCm7GHSd2fhLB8jDLw/%F0%9F%92%A0Tokens-%7C-Typography-%7C-Mobile) — для мобильного веба

## Как работают токены типографики

Есть несколько групп стилей:

- **Для наборного текста**:
  - **body** — стили для самых обычных абзацев текста
  - **body-accent** — жирные начертания body
  - **body-numeric** — стили body для табличных данных (моноширинные цифры и нулевой трекинг)
  - **body-control** — текст в контролах
  - **body-promo** — мелкие курсивные начертания (используются в Social)

- **Наборный текст с уменьшенным межстрочником**:
  - **compact** — стили для наборного текста в ячейках

- **Заголовки**:
  - **headline** — стандартные стили заголовков
  - **headline-numeric** — стили headline для табличных данных (моноширинные цифры и нулевой трекинг)
  - **headline-promo** — курсивные заголовки (используются в Social)

- **Крупные стили для привлечения внимания**:
  - **key-promo** — курсивные стили (используются в Social)


Более подробная информация в документации в [Figma/Tokens/Typography](https://www.figma.com/file/MrWp4x9dmQCNBaEVi4Arab/%F0%9F%A7%B0-Toolbox?node-id=190%3A2109
)

## Сетка шрифтов

Всего есть 8 стилей:

- `350-caption`
- `450-tagline`
- `300-x-small`
- `400-small`
- `500-medium`
- `600-large`
- `700-x-large`
- `800-xx-large`

Такое количество стилей предусмотрено и для accent, и для numeric, и для всех остальных подгрупп body.
И во всех подгруппах у аналогичных стилей одинаковый кегль и межстрочник.
Не все из них заполнены во всех подгруппах, это нормально.

## Обновление шрифтов

Чтобы обновить шрифты нужно сперва получить обновлённый JSON токенов и затем сбилдить его в css.
Для этого воспользу тесь командой `yarn color:tokens:build`

После этого поднимите patch версию ui-kit.
