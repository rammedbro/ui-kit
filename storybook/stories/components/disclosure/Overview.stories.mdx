## Описание
Испключительно нефункциональный компонент стрелки для заголовков, не имеющий состояний hover, active...
По дефолту имееет определенный цвет текста, иконки и фона, без возможности кастомизации.
В последствии компонент планируется к расширению его использования и функциональности.


В зависимости от параметра theme меняется стилистика. Если он не пришел, то выставляется дефолтное значение (DEFAULT).
Варианты theme:
- DEFAULT
- FRESH


## Примеры
`theme:`
`DEFAULT` > `FRESH`
