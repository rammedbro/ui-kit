# OzBadge

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=67%3A1149)

## Описание

Компонент отображение бейджей.
Один из параметров `text` или `icon` обязательный.

- *`text?` - однострочный, может отсутствовать только если есть `image`
- *`icon?` - компонент иконки из [Icons](http://icons.prod.a.o3.ru/?path=/story/icon--all-icons).
