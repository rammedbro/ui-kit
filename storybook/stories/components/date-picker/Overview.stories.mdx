# OzDatePicker

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=15927%3A53426)

## Описание

Компонент календарика.

Передать в модель ему можно как просто `new Date()`, так и дату в `unix time`.

Наверх календарь эммитит `unix time`.
