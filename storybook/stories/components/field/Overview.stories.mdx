# OzField

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=11077%3A22397)

## Описание

Компонент представляет собой обертку для других компонентов (в первую очередь, для [поля ввода](./?path=/story/components-ozinput-overview--page)).
Эта обертка умеет отображать следующие элементы:
* заголовок (сверху)
* список ошибок (снизу)
