## Описание

Компонент чекбокса. На состояния чекбокса можно смотреть с точки зрения заполненности и с точки зрения интерактивности.
+ Заполненность:
  * props "checked":
    + true — не заполнен
    + false — заполнен
  * props "indeterminate" (доступен для чекбокса, объединяющего группу чекбоксов, если часть внутренних чекбоксов checked = true, а часть checked = false):
    + true — группа заполнена частично
    + false - группа не заполнена
+ Интерактивность (props "disabled"):
  * true - отключён
  * false - включён

Чекбокс может иметь любую комбинацию состояний заполненности и интерактивности. Эти состояния — часть API, то есть чекбоксу можно сообщить извне, в какой паре состояний он находится.

Активация пропа usedInMobWeb добавляет псевдоэлемент, отцентрированный посередине чекбокса, с площадью 44*44px вне зависимости от размера самого чекбокса, для увеличения площади таба (клика).
