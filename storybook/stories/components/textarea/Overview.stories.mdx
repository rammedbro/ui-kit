# OzTextarea

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=16804%3A67116)

## Описание

Многострочная вариация компонента [OzInput](./?path=/docs/components-ozinput-overview--page).

Компоненту можно задать фиксированный размер в виде максимальной высоты в пикселях (`height`), куда можно передать строку вида `200px`,
или в виде максимального количества строк (`max-length`), куда нужно передать число строкой, например `maxLength="10"`.

Помимо фиксированной высоты можно указать автоподстраивание (`auto-height-by-rows`),
и тогда изначально `OzTextarea` внешне будет очень похоже на `OzInput`,
но при заполнении компонента текстом он будет увеличиваться по высоте.

По умолчанию комопонент имеет размер (`size`) **m**.
