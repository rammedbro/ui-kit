# OzTab

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=67%3A1155)

## Описание

Компонент таба.

Имеет несколько тем отображения:
- `tab` - отображение по умолчанию. Серый текст с подчеркиванием. Применяется для переключениями между вкладками внутри виджета с большим кол-вом данных в разных категориях.
- `primary` внешне выглядит как кнопка  с синим текстом на серой подложке.
- `tag` имеет скругления. Используется чаще в фильтрах.

### Counter

Каждый тип таба может иметь `counter`, число строкой (!).

У счетчика есть 2 темы отображения:
- `primary` красный счетчик
- `secondary` тёмно-серый счетчик

### TabSize

Для табов с темой 'tab' можно задать размер для таба

Есть 3 доступных размера:
- `default` размер таба по умолчанию
- `h1` размер таба со стилями текста H1
- `h2` размер таба со стилями текста H2

### Badge

Таб с темой 'tab' или 'primary' может иметь `badge`
