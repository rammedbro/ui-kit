# OzProgress

[Контракт](https://gitlab.ozon.ru/bx/contracts/-/tree/master/proto/atom/mobile/progressBar)

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=5624%3A8156)

## Описание

Компонент для отображения прогресса. Настраивается пропсами:

- `percent` задает значение прогресса
- `width` ширина линии, по умолчанию занимает вcю ширину родительского компонента
- `height` высота линии, обязательный пропс, если не передать его, линия не отобразится. По умолчанию undefined.
- `color` цвет полосы прогресса, по уполчанию `ozGreenUi`
- `backgroundColor` цвет фона полосы прогресса, по умолчанию `ozTextQuaternary`
- `time` кофигурация анимации панели: время анимации
- `bezier` кофигурация анимации панели: cubic bezier
