# OzInput

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=4235%3A6200)

## Описание

Компонент текстового поля ввода. Однострочный. Не имеет своей валидации, за исключением версии с переданным prop `mask`, который ограничивает ввод согласно маске.
Может иметь иконку, которая эмитит собственное событие клика `iconClick`.

По умолчанию имеет размер **m**  (`size="m"`).

[Многострочная версия](./?path=/docs/components-oztextarea-overview--page)

### Маска ввода

* Правила текстовой маски описаны в `./mask.js` и применяются в основном для ввода числовых значений.
* Событие `input()` возвращает два значения: 1-ое - текст без маски, 2-ое - текст с маской.
* Маска реактивна: при изменении маски, значение в самом поле ввода также изменится.
