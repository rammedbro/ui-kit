# OzTag

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=67%3A1126)

## Описание

Компонент представляет собой button, но может быть использован как ссылка если в компонент передан `href`. Тажке компонент можно дополнить `counter`'ом, который будет отображен рядом с `label`'ом. Компонент `emit`'ит события `click` при клике на `tag`. И `emit`'ит события `clickButtonClose` при клике на крестик.
