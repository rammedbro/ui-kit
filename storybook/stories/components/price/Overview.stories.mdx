# OzPrice

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=70%3A77)

[Контракт](https://gitlab.ozon.ru/bx/contracts/-/tree/master/proto/atom/mobile/price#price)

## Описание

Компонент для отображения цены товара.

## Примеры

### По умолчанию

По умолчанию цена имеет стиль `STYLE_TYPE_MEDIUM`.
Пропс `price` является обязательным.

<Canvas>
  <Story id="components-ozprice-stories--default" />
</Canvas>

### Оригинальная цена

Для акционного товара можно передать старую цену в параметр `originalPrice`.

<Canvas>
  <Story id="components-ozprice-stories--original-price" />
</Canvas>

### Цвет

С помощью параметров `priceColor` и `originalPriceColor` можно менять цвет цен.

<Canvas>
  <Story id="components-ozprice-stories--color" />
</Canvas>

### Картинка

С помощью параметра `image` устанавливается картинка слева от цены.
Параметром `imageTintColor` настраивается ее цвет.
