# OzRadioGroup

## Описание

Компонент радио-группы. Каждый элемент обладает всеми св-вами состояний [OzRadio](./?path=/docs/components-ozradio-overview--page),
но отличается тем что принимает на вход сразу массив значений.

У каждой радио-группы есть атрибут `name`,
котрый добавляется автоматически и состоит из строки `'radio'` + `index`, например `radio1`.
Также можно задать и свой кастомный `name`.
