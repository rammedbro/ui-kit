## Описание

Компоненты для отображения аватарки пользователя или его инициалов.

В зависимости от параметра `theme` меняется **размер** элемента.
Если он не пришел, то выставляется дефолтное значение (`THEME_TYPE_MEDIUM`).

Варианты `theme`:

- `THEME_TYPE_X_LARGE` (112х112)
- `THEME_TYPE_LARGE` (64х64)
- `THEME_TYPE_MEDIUM` (40х40)
- `THEME_TYPE_SMALL` (32х32)
- `THEME_TYPE_X_SMALL` (24х24)
