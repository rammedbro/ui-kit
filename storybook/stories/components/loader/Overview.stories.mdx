# OzLoader

Иконка с анимацией.

В зависимости от параметра `size` меняется **размер** элемента.
Если он не пришел, то выставляется дефолтное значение (`m`).

Заливка иконки происходит через переменную `currentColor`, т.е.
цвет соответсвует `color` элемента внутри которого она находится.

Варианты `size`:

- `s` (16x16)
- `m` (24х24)
- `l` (40х40)
