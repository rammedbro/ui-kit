# OzPortal

## Описание

Компонент `OzPortal` для рендеринга DOM вне компонента в любом месте документа.
Содержимое `OzPortal` может отображаться внутри него самого, если `OzPortalTarget` не подключен и у `OzPortal` стоит атрибут `disabled` в значении `true`.
Компонент из библиотеки [portal-vue.linusb.org](https://portal-vue.linusb.org/)
