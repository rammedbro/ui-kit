# DsImageRatio

## Описание

Компонент изображения с дополнительным функционалом. Возвращает компонент DsImage в обертке.
Имеет все те же пропсы, что и DsImage + три новых: `ratio`, `width` и `backgroundColor`.

### Примеры

#### Default
По деволту `ratio = fill`. Компонент всегда будет пропорциональным и занимать размер родителя.
В данном примере родителю задана ширина `300px`. Картинка заняла заданную ширину, а по высоте растянулась сохраняя пропорции.

  <Story id="components-dsimageratio-stories--background-color"/>
</Canvas>

Если родителю будет заданы размеры меньше размера картинки, то она обрежется, сохраняя пропорции.

  <Story id="components-dsimageratio-stories--default-with-height-container"/>
</Canvas>

#### Width
Компоненту можно напрямую задать ширину через пропс `width`.
В этом случае компонент будет иметь фиксированную ширину, а высота будет растягиваться сохраняя пропорции.
В примере задана ширина `200px`.

<Canvas>
  <Story id="components-dsimageratio-stories--width"/>
</Canvas>

#### Ratio + Width
Через пропс `ratio` можно задать пропорции изображению.
Если использовать данный пропс в связке с `width`, тогда размеры компонента будут высчитываться в скрипте и фиксироваться и изменения размеров родителя никак не повлияют на изображение.
