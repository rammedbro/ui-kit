## Описание

Компонент радио-кнопки. Редко используется одиночно. Обычно применяется в цикле.
На состояния кнопки можно смотреть с точки зрения заполненности и с точки зрения интерактивности.
+ Заполненность (props "checked"):
  * true - выбрана
  * false - не выбрана
+ Интерактивность (props "disable"):
  * true - неактивна
  * false - активна

Кнопка может иметь любую комбинацию состояний заполненности и интерактивности. Эти состояния — часть API, то есть кнопке можно сообщить извне, в какой паре состояний она находится.

Активация пропа usedInMobWeb добавляет псевдоэлемент, отцентрированный посередине кнопки, с площадью 44*44px вне зависимости от размера самой кнопки, для увеличения площади таба (клика).
