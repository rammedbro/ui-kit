# OzMessageStub

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=4336%3A5798)

## Описание

Дисклеймер (заголовок, подзаголовок) с двумя темами (`info`, `warning`) и возможностью отобразить иконку.
Отрисовать кнопку можно через передачу props (`:buttonText`, `:buttonTheme`, `@clickButton`) или через передачу
кнопки в `slot="button"`.

**ВНИМАНИЕ!**
Текущее поведение по-умолчанию для компонента: не рисовать иконку, если не передана тема.
