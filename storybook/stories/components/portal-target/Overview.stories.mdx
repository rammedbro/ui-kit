# OzPortalTarget

## Описание

Этот компонент является контейнером для любого контента, отправленного из компонента `OzPortal`.
Он только отображает полученный контент и больше ничего не делает.

Документация и описание api: [portal-vue.linusb.org/api/portal-target](https://portal-vue.linusb.org/api/portal-target.html)
