# OzPriceWithTitle

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=5693%3A0)

[Контракт](https://gitlab.ozon.ru/bx/contracts/-/tree/master/proto/atom/mobile/price#pricewithtitle-atom)

## Описание

Компонент для отображения кредитной цены.

## Примеры

### По умолчанию

По умолчанию цена имеет стиль `STYLE_TYPE_MEDIUM`.
Пропс `price` является обязательным.
Фон по умолчанию имеет цвет `ozCredit`, но его можно менять, прокинув параметр `backgroundColor`

<Canvas>
  <Story id="components-ozpricewithtitle-stories--default" />
</Canvas>

###   Title

Справа от цены можно допавить подпись, передав ее в параметр `title`. По умолчанию, цвет `ozTextSecondary`

<Canvas>
  <Story id="components-ozpricewithtitle-stories--with-title" />
</Canvas>

### Цвет

С помощью параметров `priceColor` и `titleColor` можно менять цвет цены и подписи.

<Canvas>
  <Story id="components-ozpricewithtitle-stories--color" />
</Canvas>

### Градиентный фон

С помощью параметров `gradientBackgroundColor` можно проставить фон в градиентный.
```json
{
  "startColor": "#FFB74B",
  "endColor": "#FF2F6D",
}
```
