# OzRating

[Дизайн](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=67%3A1151)

## Описание

Компонент рейтинга. Используется для отображения рейтинка товара, а так же для проставления оценки пользователем.
Имеет ряд настроек и состояний, которые настраиваются пропсами:

- `value` задает значение рейтинга
- `size` настраивает размер отборажаемого элемента
- `readOnly` отменяет кликабельность и показывает, что рейтинг находится в режиме чтения
- `withAnimation` анимирует элемент
- `altText` позволяет отразить кол-во отзывов на товар, возможен только в режиме `readOnly`
- `reviewCount` задает кол-во отзывов на товар
