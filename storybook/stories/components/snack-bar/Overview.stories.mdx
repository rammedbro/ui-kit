# OzSnackBar

[Дизайн (данный компонент использует внутри себя OzNotification и имеет такой же дизайн)](https://www.figma.com/file/F7liVaQiHzkojA2NM1ztIJ/02.-Core---UI-Components?node-id=0%3A2007)

## Описание

Компонент для вывода нотификаций на сайте. Может быть вызван глобально, с помощью notification-service.

## Примеры

### По умолчанию

Тип (type) `success`, время отображения (timeout) 5000мс, расположение (position) `BOTTOM_RIGHT`
